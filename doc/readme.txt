Matchmaking system for Fornace Studio by Gandini Simone, Vicentini Elia, Olivieri Matteo

Instructions:
-------------
1) Open Matchmaking system.py
2) Insert number of disciplines (minimum 5 disciplines)
3) Insert number of teams (minimum one more number to the disciplines)
4) When Matchmaking table.html is ready, open the file and enjoy!

Requirements:
-------------
Python 3.6
Browser web like Google Chrome or Firefox
